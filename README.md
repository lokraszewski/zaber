# Zaber 
WORK IN PROGRESS! 
The work on this project is currently ongoing and
there are no ready devices at this point.  C++ driver for the Zaber family.

## Prerequisites
* [cmake](https://cmake.org/)

## Build
```
mkdir build
cd build
cmake ..
make
```
